# মননচর্চা UI/UX

## প্রকল্পের লক্ষ্য

মননচর্চা একটি বাংলা মননশীল ডিজিটাল সাময়িকী।

এখানে পাঠক যেন সহজে সাহিত্য, ইতিহাস, দর্শন, অর্থনীতি ও সমাজবিষয়ক লেখা পড়তে পারেন।

## ডিজাইন নীতি

- Mobile First
- Minimal Design
- বাংলা টাইপোগ্রাফি অগ্রাধিকার
- দ্রুত লোডিং
- সহজ নেভিগেশন
- দীর্ঘ লেখা পড়ার উপযোগী
